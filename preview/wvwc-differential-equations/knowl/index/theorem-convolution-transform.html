Theorem 6.5.3. Convolution Theorem.

The Laplace transform distributes over convolution. In other words, if \(f\) and \(g\) are piecewise continuous functions, then
\begin{equation*} \Laplace{f(t)\ast g(t)} = \Laplace{f(t)}\Laplace{g(t)}\text{.} \end{equation*}
Equivalently, if we write \(F(s) = \Laplace{f(t)}\) and \(G(s) = \Laplace{g(t)}\text{,}\) then
\begin{equation*} \iLaplace{F(s)G(s)} = f(t)\ast g(t) = \int_{0}^{t}f(\tau)g(t-\tau)\dd{t}\text{.} \end{equation*}
