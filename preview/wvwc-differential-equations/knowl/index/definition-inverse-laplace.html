Definition 6.1.10. Inverse Laplace Transform.

The inverse Laplace transform of a function \(F(s)\text{,}\) denoted \(\mathcal{L}^{-1}\left\{F(s)\right\}\text{,}\) is a function \(f(t)\) such that \(\Laplace{f(t)} = F(s)\text{.}\)
There is an integral formula for the inverse Laplace transform as well. In particular,
\begin{equation*} \iLaplace{F(s)} = \frac{1}{2\pi i}\int_{\gamma-i\infty}^{\gamma+i\infty}F(s)e^{st}\dd{t} \end{equation*}
where \(\gamma\) is a large enough real number. We aren’t quite equipped to deal integrals such as this, so we will focus on using a table of Laplace transforms as in Table B.0.1 to compute inverse transforms.
