Example 5.1.11. A Legendre equation.

Show that
\begin{equation*} (1 - x^{2})y'' - 2xy^\prime + 2y = 0 \end{equation*}
has a series solution centered at \(0\) and then find the solution up to the coefficient of \(x^{6}\text{.}\)
Solution.
First, note that the equation can be rewritten
\begin{equation*} y'' - \frac{2x}{1 - x^{2}}y^\prime + \frac{2}{1 - x^{2}}y = 0\text{,} \end{equation*}
so we are guaranteed a series solution centered at \(x = 0\text{.}\) Furthermore, this solution has radius of convergence at least \(R = 1\text{.}\)
To find the solution, we return to the original equation and substitute \(y = \sum_{n=0}^{\infty}c_{n}x^{n}\) to get
\begin{equation*} \sum_{n\geq0}n(n-1)c_{n}x^{n-2} + \sum_{n\geq0}\brackets{-n(n-1) - 2n + 2}c_{n}x^{n} = 0 \end{equation*}
which becomes
\begin{equation*} \sum_{n\geq0}\brackets{(n + 2)(n+1)c_{n+2} - n(n-1)c_{n} - 2nc_{n} + 2c_{n}}x^{n} = 0\text{.} \end{equation*}
After a little algebra, we get the recurrence relation
\begin{equation*} c_{n+2} = \frac{n-1}{n+1}c_{n}\text{.} \end{equation*}
This recurrence is valid for \(n\geq0\text{.}\)
Now we can use the recurrence to list the first several terms of the solution:
\begin{equation*} y = c_{1}x + c_{0}(1 - x^{2} - \frac{1}{3}x^{4} - \frac{1}{5}x^{6} - \cdots) \end{equation*}
In fact,
\begin{equation*} y = c_{1}x - \sum_{n\geq0}\frac{1}{2n - 1}c_{0}x^{2n} = c_1x + C_0x\ln\parens{\frac{1+x}{1-x}}\text{.} \end{equation*}
In the last equation, \(C_0\) has been substituted for \(-\frac{c_0}{2}\text{.}\)
