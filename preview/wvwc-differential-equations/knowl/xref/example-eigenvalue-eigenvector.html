Example 4.1.9. Determining if a vector is an eigenvector.

Determine if \(\vec{v} = \begin{bmatrix}-2\\1\end{bmatrix}\) is an eigenvector of \(A = \begin{bmatrix}1\amp4\\1\amp1\end{bmatrix}\text{.}\)
Solution.
To do this, we just need to compute \(A\vec{v}:\)
\begin{equation*} A\vec{v} = \begin{bmatrix}1\amp4\\1\amp1\end{bmatrix}\begin{bmatrix}-2\\1\end{bmatrix} = \begin{bmatrix}2\\-1\end{bmatrix} = -\vec{v}\text{.} \end{equation*}
So \(\vec{v}\) is an eigenvector of \(A\) with corresponding eigenvalue \(\lambda=-1\text{.}\)
