Example 2.2.6. Repeated roots in the characteristic equation.

Find the general solution of \(y'' + 2y^\prime + y = 0\) where \(y = y(x)\text{.}\)
Solution.
We begin by solving the characteristic equation, which for this ODE is
\begin{equation*} r^{2} + 2r + 1 = 0. \end{equation*}
The only solution of this equation is \(r=-1\text{,}\) which is a repeated root. We can still get one solution of the ODE using this root, namely \(y_{1} = e^{-x}\text{,}\) but we need two linearly independent solutions in order to find the general solution. Weâ€™ll guess (and check!) that \(y_{2} = xe^{-x}\) is another solution of \(y''+2y'+y = 0\text{.}\) If we plug \(y_{2}\) into the ODE, we get
\begin{align*} y''_{2} + 2y'_{2} + y_{2} & = \underbrace{-2e^{-x} + xe^{-x}}_{y''_{2}} + \underbrace{2e^{-x} - 2xe^{-x}}_{y'_{2}} + \underbrace{xe^{-x}}_{y_{2}} \\ & = 0 \end{align*}
which shows that \(y_{2}\) is indeed a solution of the ODE. Since \(\{y_1, y_2\}\) is a linearly independent set of solutions (which we can check via the Wronskian), this means that the general solution of \(y''+2y^\prime+y=0\) is
\begin{equation*} y = c_{1}y_{1} + c_{2}y_{2} = c_{1}e^{-x} + c_{2}xe^{-x}\text{.} \end{equation*}
