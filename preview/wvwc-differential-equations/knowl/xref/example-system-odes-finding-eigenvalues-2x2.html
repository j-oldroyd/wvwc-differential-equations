Example 4.2.5. Finding eigenvalues of a \(2\times2\) matrix.

Find the eigenvalues of the matrix \(A = \begin{bmatrix}1\amp 4\\1\amp 1\end{bmatrix}\text{.}\)
Solution.
First, we need to set up the characteristic equation of \(A\text{.}\) Since
\begin{equation*} A-\lambda I = \begin{bmatrix}1-\lambda \amp 4 \\ 1 \amp 1-\lambda\end{bmatrix}\text{,} \end{equation*}
we get
\begin{equation*} \det(A-\lambda I) = (1-\lambda)^{2} - 4 = \lambda^{2}-2\lambda+3 \end{equation*}
so the characteristic equation of \(A\) is
\begin{equation*} \lambda^{2}-2\lambda+3 = 0 \end{equation*}
which has solutions \(\lambda_{1} = -1,\lambda_{2} = 3\text{.}\) So the eigenvalues of \(A\) are \(-1,3\text{.}\)
